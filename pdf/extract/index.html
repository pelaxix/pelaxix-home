How this tool works

Extract means one combined PDF.

If you enter 3-7, this tool creates one new PDF containing pages 3, 4, 5, 6, and 7. If you want separate files for each page, use Split PDF instead.

Page number format

Use commas and ranges.

Write single pages with commas, or use a dash for a range. Spaces are okay. These pages will be combined into one new PDF.

  • 1, 3, 5 creates one PDF with pages 1, 3, and 5.
  • 1-4 creates one PDF with pages 1 through 4.
  • 1-3, 7, 10-12 creates one PDF with pages 1, 2, 3, 7, 10, 11, and 12.
  • Pro move: pages stay in the order you type them. 1, 2, 4, 3 moves page 4 before page 3.
  • Pro move: duplicates are allowed. 1, 1, 2 puts page 1 in the new PDF twice.
