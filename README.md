# Pelaxix Home

Main website for pelaxix.com.

This first version is intentionally simple: plain HTML, CSS, and a tiny JavaScript file. It is meant to be easy to edit, easy to deploy, and easy to evolve while learning.

## Files

- `index.html`: homepage content and structure
- `styles.css`: visual design and responsive layout
- `script.js`: small browser-side behavior

## Cloudflare Pages settings

Use these settings when connecting the repository to Cloudflare Pages:

- Build command: leave blank
- Build output directory: `/`
- Root directory: `/`
