How this identifies the train

Schedule ID

The static schedule rows show date-specific trip IDs like 20260519-LW-1960, all sharing the same useful train pattern: LW-1960.

Train number

The tracker now defaults to 1960 because the date prefix changes, but the train pattern is consistent.

Live delay

The function checks the GO in-service train feed and tries to match 1960 against live delay data.
