How this identifies the train

Schedule ID

The static schedule rows show date-specific trip IDs like 20260519-LW-1960, all sharing the same train pattern: LW-1960.

Train number

The tracker uses 1960 as the stable target. It does not search other routes.

Live delay

The function checks the GO in-service train feed and tries to match 1960, LW-1960, or a date-prefixed trip ending in 1960.
